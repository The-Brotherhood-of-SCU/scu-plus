# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

注意：本页的更新日志应当存放一些通俗显著的更新内容，而非技术更新或者内部优化，请勿搬运git log记录。


## [Unreleased]

### Changed

- 设置页面全新改版，界面与交互体验升级
- 深色模式增强：内容不足一屏时背景色保持一致，设置页修改即时生效
- 去掉antd以及相关依赖改为自实现，大幅减少热重载和构建时间，以及打包大小

### Fixed

- 修复部分页面在深色模式下的样式问题
- 修复头像超出导航栏问题

## [2.1.0] - 2026-07-19

### Added

- 自动检查更新：可在设置中开启，检测基于 GitHub Releases，下载链接支持加速
- 界面美化设置支持选择主题点缀色
- 全新 SCU+ 品牌加载动画

### Changed

- 课表导出按钮整合：图片、JSON、ICS 三种格式统一入口
- 每学期成绩展示样式优化，使用统一的标签和边框容器

### Removed

- 移除每日一句功能及相关设置项
- 移除选课通功能及相关菜单项
- 移除重复的培养方案菜单项

## [2.0.1] - 2026-07-18

### Fixed
